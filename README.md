# expensetracker
Tech Stack

Frontend
  React (Vite)
  HTML / CSS
  Fetch API

Backend
  Java 17
  Spring Boot
  Spring Data JPA
  Hibernate
  RESTful APIs

Database
  PostgreSQL

DevOps / Tooling
  Docker
  Docker Compose
  Maven
  Git & GitHub

  Features
    Adding expenses with title, amount spent, category and date. 
    Stores data using PostgreSQL
    REST API to create, delete, and fetch expenses
    Dockerized setup for other local devs
